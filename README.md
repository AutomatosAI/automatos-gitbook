# Welcome to Automatos AI

Automatos AI is a platform for building, deploying, and managing autonomous AI agent teams. Create specialised agents, equip them with tools and knowledge, schedule their work, and let them operate — reporting back through a unified command centre.

## Quick start

### 1. Sign in

Create your account or sign in with your workspace credentials. A personal workspace is created automatically on first login.

### 2. Create your first agent

Head to [Agents](agents/README.md) and click **+ Create Agent**. Pick a template like Code Reviewer or start from scratch.

### 3. Give it tools

Go to [Tools](tools/README.md) to connect apps like GitHub, Jira, or Slack. Assign them to your agent.

### 4. Add knowledge

Upload documents in [Knowledge Bases](knowledge/README.md) so your agents can search and reference them.

### 5. Start chatting

Open [Chat](chat/README.md) and ask a question. The platform automatically routes your message to the best agent.
