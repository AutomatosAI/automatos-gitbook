# Missions

Missions are high-level objectives that span multiple tasks, agents, and time periods. Use them to track progress toward larger goals.

<!-- IMAGE: Missions tab showing active missions with progress indicators -->

## What is a mission?

A mission is a goal like:
- "Migrate authentication from v1 to v2"
- "Reduce API response time by 30%"
- "Complete Q1 security audit"

Missions group related tasks, recipes, and agent activities under a single trackable objective.

## Creating a mission

1. Click **+ New Mission** from the Missions tab
2. Define the objective
3. Set milestones and success criteria
4. Assign agents
5. Set a target completion date (optional)

## Tracking progress

Each mission shows:
- **Progress bar** — percentage toward completion
- **Milestones** — checkpoints within the mission
- **Related tasks** — linked activities contributing to the goal
- **Agent involvement** — which agents have contributed

## Mission status

| Status | Meaning |
| --- | --- |
| **Active** | In progress, agents are working on it |
| **Paused** | Temporarily on hold |
| **Completed** | All milestones met |
| **Archived** | Closed and stored for reference |

{% hint style="info" %}
Missions provide a higher-level view than individual tasks. Use them when a goal requires coordination across multiple agents over days or weeks.
{% endhint %}
