# Marketplace Agents

Browse pre-built agent configurations created by the community. Each comes ready to deploy with a tuned system prompt, recommended model, and capability set.

<!-- IMAGE: Marketplace Agents tab showing agent cards -->

## What you get

Installing a marketplace agent adds to your workspace:
- **System prompt** — tested and optimised for the agent's role
- **Model recommendation** — which LLM works best
- **Skill configuration** — git-based capabilities pre-assigned
- **Tool suggestions** — recommended integrations

## Installing an agent

1. Browse the Agents tab
2. Click an agent card to see details
3. Click **Install**
4. The agent appears on your [Agent Roster](../agents/roster.md)
5. Customise if needed

{% hint style="info" %}
Installed agents are copies — you can modify them freely without affecting the marketplace listing.
{% endhint %}

## Publishing your own agents

Share your agent configurations with the community:
1. Configure an agent in your workspace
2. Click **Publish to Marketplace** from the agent details
3. Add a description and category
4. Submit for review
