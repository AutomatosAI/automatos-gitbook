# Applications

The Applications tab shows all available external integrations powered by Composio.

<!-- IMAGE: Applications tab showing app cards with Added/Available status -->

## Browsing apps

Apps are displayed as cards in a grid. Each card shows:
- **App name** and icon
- **Category** (e.g., developer tools, productivity)
- **Tool count** — number of actions available
- **Trigger count** — number of event triggers
- **Status** — "Added" (installed) or install button

## Categories

Filter apps by category using the tab bar:

| Category | Examples |
| --- | --- |
| **All Apps** | Everything |
| **API Services** | REST APIs, webhooks |
| **Productivity** | Slack, Notion, Asana |
| **Developer Tools** | GitHub, GitLab, Linear |
| **CRM** | Salesforce, HubSpot, Pipedrive |
| **Marketing Automation** | ActiveCampaign, Mailchimp |
| **Email Marketing** | SendGrid, Benchmark Email |
| **Analytics** | Datadog, PagerDuty |

## Installing an app

1. Find the app you want
2. Click the install button on the card
3. Complete authentication (OAuth or API key)
4. The app appears as "Added" and is available for [tool assignment](../tools/assigning.md)

## Featured apps

Popular integrations include:
- **GitHub** — code repos, issues, PRs, actions
- **Slack** — messaging, channels, notifications
- **Jira** — project management, tickets
- **Stripe** — payments, subscriptions
- **Shopify** — e-commerce, orders
- **Databricks** — data engineering
- **Google Workspace** — Drive, Docs, Sheets
- **DocuSign** — document signing
- **Trello** — boards and tasks

{% hint style="info" %}
Each app provides multiple tools. GitHub alone gives you 50+ actions (create issue, review PR, search code, etc.).
{% endhint %}
