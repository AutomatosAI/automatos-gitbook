# Chat History

The chat history sidebar on the left side of the Chat page shows all your previous conversations.

<!-- IMAGE: Chat history sidebar with list of past conversations -->

## Viewing history

Click **Chat History** at the top of the sidebar to expand the conversation list. Each entry shows:
- The conversation title (auto-generated from your first message)
- A timestamp

Click any conversation to reload it and continue where you left off.

## Starting a new conversation

Click the chat icon or start typing in an empty chat to begin a fresh conversation. Previous context is not carried over — each conversation is independent.

{% hint style="info" %}
Agent memory persists across conversations. Even in a new chat, agents can recall relevant information from past interactions through the memory system.
{% endhint %}

## Tips

- **Conversations are workspace-scoped** — team members see only their own chat history
- **Long conversations** may be summarised automatically to preserve context window space
- **Agent responses** show which agent handled each message, so you can trace routing decisions

## Feedback and ratings

After any agent response, you can provide feedback:
- **Upvote/Downvote** — helps the system learn which agents give good answers
- **Corrections** — if the wrong agent responded, correct the routing

Feedback is used by the Universal Router to improve future message routing.
