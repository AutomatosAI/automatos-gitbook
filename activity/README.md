# Activity

The Activity page is your command centre. It shows what your AI workforce is doing right now, what's scheduled, and what they've reported.

<!-- IMAGE: Activity page showing Command Centre with dashboard, schedule, and agent reports -->

## Page layout

Five tabs across the top:

{% tabs %}
{% tab title="Dashboard" %}
Real-time overview of your workspace:
- **Status cards** — agents working now, channels live, tasks completed today, items needing attention
- **Schedule** — calendar view of upcoming automated routines
- **Active Now** — live tasks in progress with progress bars
- **Agent Reports** — latest standup reports from each agent

[Full guide →](dashboard.md)
{% endtab %}

{% tab title="Feed" %}
Chronological activity log. Every action, tool call, and agent response appears here in real-time.

[Full guide →](feed.md)
{% endtab %}

{% tab title="Reports" %}
Browse, filter, and grade agent reports. See what your agents found, what they did, and how well they did it.

[Full guide →](reports.md)
{% endtab %}

{% tab title="Memory" %}
Explore what your agents remember. Browse stored memories, search by topic, and manage the memory system.

[Full guide →](memory.md)
{% endtab %}

{% tab title="Missions" %}
Long-running objectives that span multiple tasks and agents. Track progress toward goals.

[Full guide →](missions.md)
{% endtab %}
{% endtabs %}

## Status cards

At the top of the Dashboard tab, four cards summarise your workspace:

| Card | What it shows |
| --- | --- |
| **Working Now** | Number of agents actively processing tasks |
| **Channels Live** | Active communication channels (Slack, email, etc.) |
| **Completed Today** | Total tasks finished in the last 24 hours |
| **Needs Attention** | Items requiring your review or intervention |

{% hint style="info" %}
Click any status card to drill down into details.
{% endhint %}
